Шаблоны управления

Шаблоны управления позволяют изменить стандартное поведение модуля вендора.
Например, нам нужно снимать FDB таблицу на одной OLT через telnet а не по snmp, поскольку не все модели одного вендора имеют одинаковые возможности.
Тогда для этой OLT нужно создать в документах шаблон с тегами system и pon_tmpl , в сам документ внести настройки, а в настройках самой OLT выбрать созданый шаблон
(Шаблон может использоваться несколькими OLT).
Для удобства все такие шаблоны можно хранить в отдельной папке.

ВНИМАНИЕ !!!
Модуль мониторинга подхватит шаблон автоматически при следующем проходе цикла!

В настоящее время, шаблоны управления поддерживаются только вендотамы ZTE и BDCOM

Все параметры и их значения вписываются как есть, без кавычек и апострофов!
Каждый параметр должен быть с новой строки!
Параметры и их значения разделяются символом " = "
Если вам не нужен какой-то параметр, его можно закомментировать символом " # " в начале строки
Перечень доступных параметров 

  •  fdb_force_telnet = 1 
  •  fdb_cmd = show mac address-table 
  •  telnet_login = admin 
  •  telnet_password = admin 
  •  telnet_timeout = 2

Параметр fdb_cmd для:
ZTE = show mac
BDCOM = show mac address-table
