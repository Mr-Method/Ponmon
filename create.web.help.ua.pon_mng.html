Шаблони керування

Шаблони керування дозволяють змінити стандартну поведінку модуля вендора.
Наприклад, нам потрібно знімати FDB таблицю на одній OLT через telnet а не по snmp, оскільки не всі моделі одного вендора мають одинакові можливості.
Тоді для цієї OLT потрібно створити в документах шаблон з тегами system та pon_tmpl, в сам документ внести налаштування, а в налаштуваннях самої OLT вибрати створений шаблон
(Шаблон може використовуватись декількома OLT ).
Для зручності, всі такі шаблони можна зберігати в окремій папці.

УВАГА!!!
Модуль моніторингу підхопить шаблон автоматично при наступному проході циклу!

На даний час, шаблони керування підтримуються тільки вендорами ZTE та BDCOM

Всі параметри і їх значення вписуються як є, без лапок та апострофів!
Кожен параметр має бути з нового рядка!
Параметри і їх значення розділяються символом "="
Якщо вам не потрібний якийсь параметр, його можна закоментувати символом "#" на початку рядка
Перелік доступних параметрів:

  • fdb_force_telnet = 1
  • fdb_cmd = show mac address-table
  • telnet_login = admin
  • telnet_password = admin
  • telnet_timeout = 2

Параметр fdb_cmd для:
ZTE = show mac
BDCOM = show mac address-table
